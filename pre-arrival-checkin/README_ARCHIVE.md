# Archive copy (not used by Maven or Docker)

The **live application** is built only from **`src/` at the repository root** (next to `pom.xml`). This folder is kept as a **reference backup** (guides, older snippets, template copies).

Do **not** edit files here expecting Render or `mvn package` to pick them up—edit **`src/main/...`** at the repo root instead.
